# 小墩子故事屋 · 里程碑与排期（2-4周）

总体：预算充足、无需额外算力；目标周期 3 周交付正式版（2 周冲刺版可行，4 周含打磨冗余）。

里程碑
- W1（原型与骨架）
  - App 基础框架（View + Navigation）、主题与品牌草案
  - 数据模型与本地库（Room）、播放前台服务 + MediaSession
  - TTS 播放、后台播放、进度记忆、定时器
  - txt 导入（编码识别、段落解析、封面推断）
  - 预置内容 50 篇（含标签/适龄/价值观）
  - GitHub Actions 工作流产出 APK（未签名或测试签名）
  - Demo 检视与验收清单

- W2（功能完整 + 300 篇内容）
  - epub 导入（OPF/NCX/插图/封面解析）
  - 插画轮播（按段落/页）、手势与自动翻页
  - 标签检索、适龄/主题推荐首页
  - 预置内容扩充至 ≥300 篇（去重 + 价值观检查 + 元数据齐全）
  - 家长区设置、导入/在线源配置
  - 多机型兼容性测试（minSdk 24 设备）
  - RC-1 内测包

- W3（联网上线 + 文档与发布）
  - 远程源（remote-feed）与 generator-agent 接入（自动发布）
  - 推荐与离线索引优化、错误处理与稳定性
  - 安装/部署文档完善，首启引导与示例导入
  - 性能优化与耗电评测，崩溃与严格权限流程（Android 13+）
  - 正式签名/发布产物（可选）、RC-2 验收

- W4（可选打磨）
  - 视觉细节与动画润色、包体与冷启动优化
  - 内容再扩充与分龄推荐精细化
  - 商店上架物料（如需）

关键检查点
- 每周末 Demo：功能走查 + 指标检查（启动时长、TTS 稳定、导入成功率）
- 内容验收：W2 末完成 ≥300 篇，重复率可控

技术参数（锁定）
- minSdk 24、targetSdk 35；Kotlin + View 体系；TTS + MediaSession；Room；Retrofit/OkHttp

风险与缓解
- 内容规模：采用模板化生成 + 公有领域改写 + 去重，滚动补全
- 插画资源：采用风格统一的小插画集 + 自动裁切与分段锚点

交付物
- APK（Actions Artifact）、源代码、安装/部署文档、变更与里程碑报告
